#以太坊技术路线图
以太坊技术路线图首次批露于2015年3月，主要包括四个发布阶段

##Frontier（前沿）‘

2015年7月30日，Frontier的创始区块产生。Frontier是以太坊的第一版，提供了基本的命令行功能，用户可以开采以太币以及上传并执行合约。

##Homestead（家园）

首个以太坊的生产版本，发布于2016年3月14日。家园仍然是主打命令行界面，但是构成了以太坊技术的首个商业化迭代。家园自动在区块数1，150，000被自动引入以太坊区块链上。
